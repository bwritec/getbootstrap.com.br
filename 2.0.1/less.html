Construído com LESS

Por que LESS ?

Bootstrap é feito com LESS em sua essência, uma linguagem de folha de estilo dinâmica, Isso torna o desenvolvimento de CSS baseado em sistemas mais rápido, fácil e divertido.

O que está incluído ?

Como uma extensão do CSS, LESS inclui variáveis, mixins para trechos de código reutilizáveis, operações para matemática simples, aninhamento e até funções de cores.

Saber mais

Visite o site oficial em http://lesscss.org/ para saber mais.

Variáveis

Gerenciar cores e valores de pixel em CSS pode ser um pouco trabalhoso, geralmente cheio de copiar e colar. Porém, não com LESS - atribua cores ou valores de pixel como variáveis e altere-os uma vez.

Mixins

Essas três declarações de border-radius que você precisa fazer no velho CSS normal ? Agora eles estão reduzidos a uma linha com a ajuda de mixins, trechos de código que você pode reutilizar em qualquer lugar.

Operações

Torne sua régua, líder e mais flexível, fazendo as contas dinamicamente com as operações. Multiplique, divida, adicione e subtraia para obter a sanidade do CSS.

Bootstrap mixins

Sobre mixins

Mixins básicos

Um mixin básico é essencialmente uma inclusão ou parcial de um trecho de CSS. Eles são escritos como uma classe CSS e podem ser chamados em qualquer lugar. 

.element
{
    .clearfix();
}

Mixins paramétricos

Um mixin paramétrico é como um mixin básico, mas também aceita parâmetros (daí o nome) com valores padrão opcionais. 

.element
{
    .border-radius(4px);
}

Adicione facilmente o seu próprio

Quase todos os mixins do Bootstrap são armazenados em mixins.less, um maravilhoso arquivo utilitário .less que permite usar um mixin em qualquer um dos arquivos .less do kit de ferramentas.

Então, vá em frente e use os existentes ou fique à vontade para adicionar os seus conforme necessário.

Compilando Bootstrap com LESS

Observação: Se estiver enviando uma solicitação pull ao Git com CSS modificado, você deve recompilar o CSS por meio de qualquer um desses métodos.

Ferramentas para compilar

Node com makefile

Instale o compilador de linha de comando LESS globalmente com npm executando o seguinte comando: 

$ npm install -g less

Depois de instalado basta executar make da raiz do seu diretório de inicialização e está tudo pronto.

Além disso, se você tiver watchr instalado, você pode executar make watch para que o bootstrap seja reconstruído automaticamente toda vez que você editar um arquivo na biblioteca de bootstrap (isso não é necessário, apenas um método conveniente).

Linha de comando

Instale a ferramenta de linha de comando LESS via Node e execute o seguinte comando: 

$ lessc ./lib/bootstrap.less > bootstrap.css

Certifique-se de incluir --compress nesse comando se você estiver tentando salvar alguns bytes !

Javascript

Baixar o Less.js mais recente e inclua o caminho para ele (e Bootstrap) no <head>. 

<link rel="stylesheet/less" href="/path/to/bootstrap.less">
<script src="/path/to/less.js"></script>

Para recompilar os arquivos .less, basta salvá-los e recarregar sua página. Less.js os compila e os armazena no armazenamento local.
