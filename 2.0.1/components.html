Grupos de botões Junte-se aos botões para obter mais funcionalidades semelhantes às da barra de ferramentas

Grupos de botões

Use grupos de botões para unir vários botões como um componente composto. Construa-os com uma série de elementos <a> ou <button>.

Melhores Práticas

Recomendamos as seguintes diretrizes para usar grupos de botões e barras de ferramentas:

  • Sempre use o mesmo elemento em um único grupo de botões, <a> ou <button>.
  • Não misture botões de cores diferentes no mesmo grupo de botões.
  • Use ícones além ou em vez de texto, mas certifique-se de incluir texto alternativo e de título quando apropriado.

Relacionado Grupos de botões com menus suspensos (veja abaixo) devem ser chamados separadamente e sempre incluir um cursor suspenso para indicar o comportamento pretendido.

Exemplo padrão

Veja como o HTML procura um grupo de botões padrão criado com botões de tag âncora:

    <a class="btn" href="#">
        1
    </a>

    <a class="btn" href="#">
        2
    </a>

    <a class="btn" href="#">
        3
    </a>
</div>

Padrões leves Mesma marcação, classes diferentes

Classe base poderosa

Quando usar

Nav tabulável Dê vida às tabs via javascript

O que está incluído

Dê vida às suas guias com um plug-in simples para alternar entre o conteúdo por meio das tabs. Bootstrap integra tabs tabuláveis em quatro estilos: mais alto (padrão), direita, mais baixo e esquerda.

Alterar entre eles é fácil e requer apenas alterar muito pouca marcação.

A navbar requer apenas alguns divs para estruturá-la bem para exibição estática ou fixa. 

<div class="navbar">
    <div class="navbar-inner">
        <div class="container">
            ...
        </div>

Navbar fixo

Para fixar a navbar no topo da janela de visualização, adicione .navbar-fixed-top para o div mais externo, .navbar. 

<div class="navbar navbar-fixed-top">
    ...
</div>

No seu CSS, você também precisará levar em conta a sobreposição que causa adicionando 40px ou mais de preenchimento ao seu <body>. Certifique-se de adicionar isso após o CSS principal do Bootstrap e antes do CSS responsivo opcional.

Nome da brand

Um link simples para mostrar sua marca ou nome de projeto requer apenas uma tag âncora. 

<a class="brand" href="#">
    Nome do projeto
</a>

Formulário de busca

Os formulários de busca recebem estilos personalizados na navbar com a classe .navbar-search. Incluir .pull-left ou .pull-right no form para alinhá-lo. 

<form class="navbar-search pull-left">
    <input type="text" class="search-query" placeholder="Buscar">
</form>

Variação responsiva opcional

Dependendo da quantidade de conteúdo na sua navbar, você pode querer implementar as opções responsivas. Para fazer isso, envolva seu conteúdo de navegação em um div contendo, .nav-collapse.collapse, e adicione o botão de alternância da navbar, .btn-navbar. 

<div class="navbar">
    <div class="navbar-inner">
        <div class="container">
            <!--
              - .btn-navbar é usado como alternância para o conteúdo
              - recolhido da navbar.
             -->
            <a class="btn btn-navbar" data-toggle="collapse" data-target=".nav-collapse">
                <span class="icon-bar"></span>
                <span class="icon-bar"></span>
                <span class="icon-bar"></span>
            </a>

            <!--
              - Certifique-se de deixar a marca lá fora se quiser
              - que ela seja mostrada.
             -->
            <a class="brand" href="#">
                Nome do projeto
            </a>

            <!--
              - Tudo o que você deseja ocultar em 940px ou menos,
              - coloque aqui.
             -->
            <div class="nav-collapse">
                <!--
                  - .nav, .navbar-search, .navbar-form, etc.
                 -->
            </div>
        </div>
    </div>
</div>
Atenção ! A navbar responsiva requer o plugin collapse.

Por que usar thumbnails

Thumbnails (anteriormente .media-grid até v1.4) são ótimos para grades de fotos ou vídeos, resultados de pesquisa de imagens, produtos de varejo, portfólios e muito mais. Podem ser links ou conteúdo estático.

Marcação simples e flexível

Marcação de thumbnail é simples - um ul com qualquer número de elementos li é tudo o que é necessário. Também é super flexível, permitindo qualquer tipo de conteúdo com um pouco mais de marcação para agrupar seu conteúdo.

Usa tamanhos de coluna da régua

Por último, o componente de miniaturas usa classes de sistema de régua existentes - como .span2 ou .span3 - para controle das dimensões das thumbnails.

A marcação

Conforme mencionado anteriormente, a marcação necessária para thumbnails é leve e direta. Aqui está uma olhada na configuração padrão para imagens vinculadas: 

<ul class="thumbnails">
    <li class="span3">
        <a href="#" class="thumbnail">
            <img src="./assets/260x180.png" alt="">
        </a>
    </li>

    ...
</ul>

Para conteúdo HTML personalizado em thumbnails, a marcação muda ligeiramente. Para permitir conteúdo em nível de bloco em qualquer lugar, trocamos o <a> para um <div> igual a: 

<ul class="thumbnails">
    <li class="span3">
        <div class="thumbnail">
            <img src="./assets/img/260x180.png" alt="">

            <h5>
                Label do thumbnail
            </h5>

            <p>
                Caption da thumbnail bem aqui...
            </p>
        </div>
    </li>

    ...
</ul>

Alertas Estilos para mensagens de sucesso, aviso e erro

Padrões leves

Classe base reescrita

Com o Bootstrap 2, simplificamos a classe base: .alert em vez de .alert-message. Também reduzimos a marcação mínima exigida - <p> não é obrigatório por padrão, apenas o <div> exterior.

Mensagem de alerta única

Para um componente mais durável e com menos código, removemos a aparência diferenciadora para alertas de bloco, mensagens que vêm com mais preenchimento e normalmente mais texto. A classe também mudou para .alert-block.

Vai muito bem com javascript

Bootstrap vem com um ótimo plugin jQuery que suporta mensagens de alerta, tornando rápido e fácil descartá-las.

Obter o plugin »

Exemplo de alertas

Envolva sua mensagem e um ícone opcional de fechamento em uma div com classe simples.

× Aviso ! É melhor verificar você mesmo, você não está com uma aparência muito boa. 
<div class="alert">
    <a class="close" data-dismiss="alert">
        ×
    </a>

    <strong>
        Aviso !
    </strong>

    É melhor verificar você mesmo, você não está com
    uma aparência muito boa.
</div>

Estenda facilmente a mensagem de alerta padrão com duas classes opcionais: .alert-block para mais preenchimento e controles de texto e .alert-heading para um título correspondente.

×

Aviso !

É melhor verificar você mesmo, você não está com uma aparência muito boa. Conversamos, diversidade obrigado bem tudo diversidade promessa aprender prometer eloquência esperança prazer. 

<div class="alert alert-block">
    <a class="close" data-dismiss="alert">
        ×
    </a>

    <h4 class="alert-heading">
        Aviso !
    </h4>

    É melhor verificar você mesmo, você não está...
</div>

Barras de progresso Para carregamento, redirecionamento ou status de ação

Exemplos e marcação

Basic

Barra de progresso padrão com gradiente vertical. 

<div class="progress">
    <div class="bar" style="width: 60%;">
    </div>
</div>

Striped

Usa um gradiente para criar um efeito listrado. 

<div class="progress progress-info
     progress-striped">
  <div class="bar"
       style="width: 20%;"></div>
</div>

Animado

Pega o exemplo listrado e o anima. 

<div class="progress progress-danger progress-striped active">
    <div class="bar" style="width: 40%;">
    </div>
</div>

Opções e suporte ao navegador

Cores adicionais

As barras de progresso utilizam alguns dos mesmos nomes de classe dos botões e alertas para estilos semelhantes.

  • .progress-info
  • .progress-success
  • .progress-danger

Alternativamente, você pode personalizar os arquivos LESS e criar suas próprias cores e tamanhos.

Comportamento

As barras de progresso usam transições CSS3, portanto, se você ajustar dinamicamente a largura via javascript, ela será redimensionada suavemente.

Se você usar a classe .active, sua barra de progresso .progress-striped animarão as listras da esquerda para a direita.

Suporte ao navegador

As barras de progresso usam gradientes, transições e animações CSS3 para obter todos os seus efeitos. Esses recursos não são suportados no IE7-8 ou em versões anteriores do Firefox.

O Opera não oferece suporte a animações no momento.
