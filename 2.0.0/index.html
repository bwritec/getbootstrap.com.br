Projetado para todos, em qualquer lugar.

Precisa de motivos para amar o Bootstrap ? Não procure mais.

Construído para e por pessoas

Assim como você, adoramos criar produtos incríveis na web. Amamos tanto isso que decidimos ajudar pessoas como nós a fazer isso de maneira mais fácil, melhor e mais rápida. Bootstrap foi criado para você.

Para todos os níveis de habilidade

Bootstrap foi projetado para ajudar pessoas de todos os níveis de habilidade - designer ou desenvolvedor, avançado ou iniciante. Use-o como um kit completo ou para iniciar algo mais complexo.

Cross-everything

Originalmente desenvolvido pensando apenas nos navegadores modernos, o Bootstrap evoluiu para incluir suporte para todos os principais navegadores (até mesmo o IE7!) E, com o Bootstrap 2, também para tablets e smartphones.

Régua de 12 colunas

Os sistemas de régua não são tudo, mas ter um sistema durável e flexível no centro do seu trabalho pode tornar o desenvolvimento muito mais simples. Use nossas classes de régua integradas ou crie a sua própria.

Design responsivo

Com o Bootstrap 2, nos tornamos totalmente responsivos. Nossos componentes são dimensionados de acordo com uma variedade de resoluções e dispositivos para fornecer uma experiência consistente, não importa o que aconteça.

Documentos do guia de estilo

Ao contrário de outros kits de ferramentas front-end, o Bootstrap foi projetado principalmente como um guia de estilo para documentar não apenas nossos recursos, mas também práticas recomendadas e exemplos codificados.

Biblioteca crescente

Apesar de ter apenas 10kb (gzipado), Bootstrap é um dos kits de ferramentas de front-end mais completos que existem, com dezenas de componentes totalmente funcionais prontos para serem usados.

Plug-ins jQuery personalizados

De que adianta um componente de design incrível sem interações fáceis de usar, adequadas e extensíveis ? Com o Bootstrap, você obtém plug-ins jQuery personalizados para dar funcionalidades aos seus projetos.

Construído em LESS

Onde o vanilla CSS falha, LESS se destaca. Variáveis, aninhamentos, operações e mixins em LESS tornam a codificação CSS mais rápida e eficiente com sobrecarga mínima.
