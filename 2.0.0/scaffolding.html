O sistema de régua padrão fornecido como parte do Bootstrap é uma régua de 940px de largura e 12 colunas.

Ele também possui quatro variações responsivas para vários dispositivos e resoluções: telefone, retrato de tablet, paisagem de mesa e desktops pequenos e desktops widescreen grandes.

Colunas de aninhamento

Com o sistema de régua estático (não fluido) do Bootstrap, o aninhamento é fácil. Para aninhar seu conteúdo, basta adicionar um novo .row e conjunto de colunas .span* dentro de uma coluna .span* existente.

Personalização da régua

Variável Valor padrão Descrição
@gridColumns 12 Numero de colunas
@gridColumnWidth 60px Largura de cada coluna
@gridGutterWidth 20px Espaço negativo entre colunas
@siteWidth Soma calculada de todas as colunas e gutters Conta o número de colunas e gutters para definir a largura do mixin .container-fixed().

Variáveis em LESS

Integradas ao Bootstrap estão algumas variáveis para personalizar o sistema de régua padrão de 940px, documentado acima. Todas as variáveis da régua são armazenadas em variables.less.

Como personalizar

Modificar a régua significa alterar as três variáveis @grid* e recompilar o Bootstrap. Altere as variáveis da régua em variables.less e use uma das quatro formas documentadas para recompilar. Se você estiver adicionando mais colunas, certifique-se de adicionar o CSS para aquelas em grid.less.

Permanecendo responsivo

A personalização da régua funciona apenas no nível padrão, a régua de 940px. Para manter os aspectos responsivos do Bootstrap, você também terá que personalizar as réguas em responsive.less.

Layouts Modelos básicos para criar páginas da web

Layout fixo

O layout centralizado padrão e simples de 940 pixels de largura para praticamente qualquer site ou página fornecido por um único <div class="container">. 

<body>
    <div class="container">
        ...
    </div>
</body>

Layout fluido

<div class="container-fluid"> oferece estrutura de página flexível, larguras mínima e máxima e uma barra lateral esquerda. É ótimo para aplicativos e documentos. 

<div class="container-fluid">
    <div class="row-fluid">
        <div class="span2">
            <!--Sidebar content-->
        </div>

        <div class="span10">
            <!--Body content-->
        </div>
    </div>
</div>

Design responsivo Consultas de mídia para vários dispositivos e resoluções

Dispositivos suportados

Bootstrap oferece suporte a diversas consultas de mídia para ajudar a tornar seus projetos mais apropriados em diferentes dispositivos e resoluções de tela. Aqui está o que está incluído:

Label Largura do layout Largura da coluna Largura do gutter
Smartphones 480px e abaixo Colunas fluidas, sem larguras fixas
Tablets em modo retrato 480px para 768px Colunas fluidas, sem larguras fixas
Tablets em modo paisagem 768px para 940px 44px 20px
Padrão 940px ou mais 60px 20px
Display grande 1210px ou mais 70px 30px

O que eles fazem

As consultas de mídia permitem CSS personalizado com base em diversas condições - proporções, larguras, tipo de exibição etc. - mas geralmente se concentram em min-width e max-width.

  • Modifique a largura da coluna em nossa régua
  • Empilhe elementos em vez de flutuar sempre que necessário
  • Redimensione os títulos e o texto para serem mais apropriados aos dispositivos

Usando as consultas de mídia

O Bootstrap não inclui automaticamente essas consultas de mídia, mas entendê-las e adicioná-las é muito fácil e requer configuração mínima. Você tem algumas opções para incluir os recursos responsivos do Bootstrap:

  1. Use a versão responsiva compilada, bootstrap-responsive.css
  2. Adicione @import "responsive.less" e recompile o Bootstrap
  3. Modifique e recompile responsive.less como um arquivo separado

Por que não apenas incluí-lo ? Verdade seja dita, nem tudo precisa ser responsivo. Em vez de encorajar os desenvolvedores a remover esse recurso, achamos melhor ativá-lo. 

    /**
     * Telefones em modo paisagem e para baixo.
     */
    @media (max-width: 480px)
    {
        ...
    }

    /**
     * Telefone em modo paisagem para tablet em modo retrato.
     */
    @media (max-width: 768px)
    {
        ...
    }

    /**
     * Tablet em modo retrato para desktop em modo paisagem.
     */
    @media (min-width: 768px) and (max-width: 940px)
    {
        ...
    }

    /**
     * Desktop em modo paisagem.
     */
    @media (min-width: 1200px)
    {
        ...
    }
